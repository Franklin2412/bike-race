# bike-race

A retro-style pseudo-3D racing game inspired by Road Rash, built for GitHub Pages.

## Features
- **Pseudo-3D Engine**: Smooth curves and hills using 16-bit math.
- **Dynamic Obstacles**: Moving cars and pedestrians.
- **Trash Mode**: Smash trash bags for bonus points!
- **Purely Static**: No build steps, hosted directly on GitHub Pages.

## How to Play
1. Open [index.html](https://Franklin2412.github.io/bike-race/) (after deployment).
2. Click **START RACE**.
3. Use **Arrow Keys** to drive.

## Controls
- `Up`: Accelerate
- `Down`: Brake
- `Left/Right`: Steering
